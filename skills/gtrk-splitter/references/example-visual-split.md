# 全片总览

- 三幕结构：第一幕用身体参与和日常痛感把观众拖进镜像认同；第二幕用“过拟合”翻译强迫性重复并一路坠入实在界；第三幕用“双重下降”完成容器反转，再以“幽灵堵车”回扣收尾。
- 容器贯穿线：`seed` 于“荒诞的过拟合”；`expand/translate` 于 AI 概念解释和心理学映射；`flip` 于双重下降；`callback` 于幽灵堵车。
- 必须回扣的意象：`掐虎口`、`过拟合`、`幽灵堵车`
- 静音/停顿节点：`我们，一直在自己掐自己的虎口` 之后；`是不是就能脱离这种循环` 之后；`如果一切都是潜意识的剧本` 之后

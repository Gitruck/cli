# 种子 · 家居好物带货栏目（「李姐严选」削减版）

一人主理的家居好物口播带货号。访谈收敛出的家族：**1 个 skill（单产线不切）+ 词表并入 SKILL.md**。

## 组织维度（全部来自栏目主自己的语言）

- **Step 1 = 痛点**：她的第一判断是「这货到底解决啥难受」——痛点必须**能拍出来**（一滩酱油/一声「咔」/一个乱抽屉），开头三秒戳它；且次序铁定「先戳痛点再亮卖点，反了人就划走」。痛点找不着，这单不接。
- **第二维度 = 上手**：信任感的唯一来源是「真人上手」——手 + 货 + 使劲的动作同画面，环境必须是过日子的地方；演不出来的卖点降级为嘴上带过，不给镜头。
- **第三维度 = 砸价**：价格是一个「出场动作」，不是一条信息——三选一判定（手上动作 / 憋一下再喊 / 全屏只剩一个大数），且砸之前全篇不许漏价。
- **第四维度 = 现在就买的理由**：结尾闭环，最后一句必须给一个说得出口的理由（马上用得上的场景 / 限时 / 赠品），不许只说「喜欢的拍」。
- **负面护栏 = 廉价感**：她说不出「风格是什么」，但对「什么显得廉价」秒答——七条禁令（综艺花字/满屏贴纸/「duang」类音效/抠图假背景/磨皮到看不清货/机器人配音/价格缩在角落小字）是全套规矩里最硬的资产。

## 铁律（禁令先行的产物，节选）

- 廉价感七条禁令，中一条即重剪。
- 前三秒只许出现「难受」：不许品牌、自我介绍、价格、参数。
- 砸价三选一，占零样 = 念价格，返工；砸前漏价 = 白砸。
- 字幕最多两行大白字；真人原声，带口音也不换机器配音。
- 每条铁律配自检问句（例：「只看前三秒，看没看见一个具体的难受？」「砸价那一秒，三样占了几样？」「最后五秒能不能一句话复述那个现在就买的理由？」）。

## 家族切分

单产线不切家族：只有一条产线（带货片子），封面从片中截帧、不单独生产 → 1 个 skill 足够。`produces` = 自造串「带货台本」，查 handoff 映射表未命中注册集 → 无契约绑定，登记加 `routing:"none"`。

## 词表（并入 SKILL.md 一节）

已沉淀自造词，但家族仅 1 个 skill、无跨文件漂移面 → 词表并入 SKILL.md，并写明「出现第二个 skill 时整节迁出为 `_shared` 词表，各 skill 引用不复制」。词条（全部她的原词）：

**痛点 / 卖点 / 上手 / 砸价 / 信任感 / 廉价感 / 划走 / 台本**（产物名）。

## 自检结构

自检清单逐条对应铁律，问句全部可判定、且用她的词（「砸价那一秒三样占了几样？」而非「价格强调是否充分？」）。给不出判定阈值的主张（「亮堂」「干净」「东西少」——她的原话「说不上数，凭眼睛」）降级为「倾向」，单列一节，不进铁律、不进自检硬条目。

## 这个种子教你什么

维度名（痛点/上手/砸价/廉价感/划走）**只属于这个栏目**。可迁移的是形态：①**负面维度可以当第一等组织维度**——说不出「我是什么」但秒答「什么绝对不行」的创作者，禁令清单就是骨架；②**次序本身可以是铁律**（先痛后卖），不只是内容主张；③判定不了的主张降级为倾向，绝不硬凑阈值；④单产线绝不硬拆家族；⑤`produces` 写自造串完全合法——框架静默登记，管线外加 `routing:"none"` 即可，不逼创作者学管线词汇。
